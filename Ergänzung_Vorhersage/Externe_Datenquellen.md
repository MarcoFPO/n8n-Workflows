# Externe Datenquellen (Kostenlos für private Nutzung)

Eine Liste von externen Datenquellen, die für private Forschungszwecke kostenlos sind und eine tägliche Abfrage ermöglichen.

| Datenkategorie | Datenquelle | Abdeckung | Technische Referenzen & Beispiel-Code |
| :--- | :--- | :--- | :--- |
| **Kurs- & Volumendaten** | **Yahoo Finance** | Sehr gut (International) | **- Webseite:** [finance.yahoo.com](https://finance.yahoo.com)<br>**- Python-Bibliothek:** `yfinance`<br>**- Doku:** [pypi.org/project/yfinance](https://pypi.org/project/yfinance/)<br>```python<br>import yfinance as yf<br># Ticker für VW (XETRA) abrufen<br>data = yf.download('VOW3.DE', start='2023-01-01')<br>print(data.head())<br>``` |
| **Kurs- & Fundamentaldaten** | **Alpha Vantage** | Gut (International) | **- Webseite:** [alphavantage.co](https://www.alphavantage.co)<br>**- API-Doku:** [alphavantage.co/documentation](https://www.alphavantage.co/documentation/)<br>**- Zugriff:** REST API (z.B. mit `requests`)<br>```python<br>import requests<br>API_KEY = 'DEIN_KOSTENLOSER_API_KEY'<br>url = f'https://www.alphavantage.co/query?function=OVERVIEW&symbol=IBM&apikey={API_KEY}'<br>r = requests.get(url)<br>data = r.json()<br>print(data['RevenuePerShareTTM'])<br>``` |
| **Sentiment (Nachrichten)** | **NewsAPI** | Gut (International) | **- Webseite:** [newsapi.org](https://newsapi.org)<br>**- API-Doku:** [newsapi.org/docs](https://newsapi.org/docs)<br>**- Zugriff:** REST API (z.B. mit `requests`)<br>```python<br>import requests<br>API_KEY = 'DEIN_KOSTENLOSER_API_KEY'<br>url = f'https://newsapi.org/v2/everything?q=Siemens&language=de&apiKey={API_KEY}'<br>r = requests.get(url)<br>data = r.json()<br>print(data['articles'][0]['title'])<br>``` |
| **Sentiment (Social Media)** | **Reddit API** | Gut (International) | **- Webseite:** [reddit.com/dev/api](https://www.reddit.com/dev/api/)<br>**- Python-Bibliothek:** `PRAW`<br>**- Doku:** [praw.readthedocs.io](https://praw.readthedocs.io/en/latest/)<br>```python<br>import praw<br># Authentifizierungs-Details erforderlich<br>reddit = praw.Reddit(client_id='...', client_secret='...', user_agent='...')<br>subreddit = reddit.subreddit('investing')<br>for post in subreddit.hot(limit=5):<br>    print(post.title)<br>``` |
| **Makroökonomische Daten (USA & Global)** | **FRED** | Exzellent (US), Gut (Global) | **- Webseite:** [fred.stlouisfed.org](https://fred.stlouisfed.org)<br>**- Python-Bibliothek:** `pandas-datareader`<br>**- Doku:** [pandas-datareader.readthedocs.io](https://pandas-datareader.readthedocs.io/en/latest/remote_data.html#remote-data-fred)<br>```python<br>import pandas_datareader.data as web<br># US Volatility Index (VIX)<br>vix = web.DataReader('VIXCLS', 'fred', start='2023-01-01')<br>print(vix.tail())<br>``` |
| **Makroökonomische Daten (Europa)** | **Eurostat** | Exzellent (Europa) | **- Webseite:** [ec.europa.eu/eurostat](https://ec.europa.eu/eurostat)<br>**- Python-Bibliothek:** `pandas-datareader`<br>**- Doku:** [pandas-datareader.readthedocs.io](https://pandas-datareader.readthedocs.io/en/latest/remote_data.html#remote-data-eurostat)<br>```python<br>import pandas_datareader.data as web<br># Inflationsrate (HICP) Euro-Zone<br>hicp = web.DataReader('prc_hicp_manr', 'eurostat', start='2023-01-01')<br>print(hicp.tail())<br>``` |
| **Makroökonomische Daten (Industrieländer)** | **OECD Data** | Exzellent (Industrieländer) | **- Webseite:** [data.oecd.org](https://data.oecd.org)<br>**- Python-Bibliothek:** `pandas-datareader`<br>**- Doku:** [pandas-datareader.readthedocs.io](https://pandas-datareader.readthedocs.io/en/latest/remote_data.html#remote-data-oecd)<br>```python<br>import pandas_datareader.data as web<br># Verbrauchervertrauen Deutschland (DEU)<br>cci = web.DataReader('CSCICP02', 'oecd', start='2023', country='DEU')<br>print(cci.tail())<br>``` |
| **Makroökonomische Daten (Weltweit)** | **World Bank** | Exzellent (Weltweit) | **- Webseite:** [data.worldbank.org](https://data.worldbank.org)<br>**- Python-Bibliothek:** `pandas-datareader`<br>**- Doku:** [pandas-datareader.readthedocs.io](https://pandas-datareader.readthedocs.io/en/latest/remote_data.html#remote-data-wb)<br>```python<br>from pandas_datareader import wb<br># GDP für Deutschland und USA<br>gdp = wb.download(indicator='NY.GDP.MKTP.CD', country=['DE', 'US'], start=2018, end=2022)<br>print(gdp)<br>``` |
